# humaneforms

Form fields that never reject a valid human — correct handling of names, emails,
phone numbers, postal addresses, and dates, out of the box.

## Packages

- **`@humaneforms/core`** — framework-agnostic validation + normalization plus the
  edge-case corpus. Zero UI dependencies.
- **`@humaneforms/react`** — React components and hooks built on `core` (free tier).

## Getting started

```bash
pnpm install
pnpm build
pnpm test
```
